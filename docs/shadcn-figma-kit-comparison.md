## 범례

- **Variables**: Figma Variables 사용 여부 (다크모드/토큰 전환 가능)
- **다크모드**: Variable mode로 Light/Dark 전환 지원
- **1:1 매핑**: shadcn 코드의 variant/size/state와 Figma prop 일치
- **Blocks**: 재사용 가능한 UI 블록 (Hero, Pricing, Dashboard 등)
- **컬러**: Variables로 관리되는 색상 시스템
- **타이포**: Variables로 관리되는 폰트 시스템
- **Auto Layout**: 컴포넌트에 Hug/Fill, Gap, Padding 적용 여부

### 경고 표기 (⚠️)
- **prop명 다름**: shadcn은 variant/size, Kit은 type/state 등 다른 이름 사용
- **Styles**: Variables가 아닌 Color Styles만 사용 (다크모드 자동 전환 불가)
