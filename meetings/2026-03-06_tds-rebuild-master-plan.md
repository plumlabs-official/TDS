### [ENGINEERING LEAD]

**의존성 그래프:**

```
Variables (Color)
    ↓
Variables (Spacing, Radius)
    ↓
Text Styles (Typography)
    ↓
Effect Styles (Shadow)
    ↓
Components (Button, Input...)
    ↓
Patterns (Card, Dialog...)
```

**Build vs Buy:**
- Variable 구조 → Buy (shadcn 그대로)
- Color 값 → Build (Tryve 교체)
- Typography 구조 → Buy (shadcn 그대로)
- Font Family → Build (Tryve 폰트)
